# Вклад в проект

## Правила
- Соблюдайте стиль TypeScript и принятые паттерны
- Малые PR, фокус на конкретных задачах
- Обновляйте документацию при изменениях API или моделей

## Запуск локально
- Настройте `.env`
- `npm install`
- `npm run dev`
